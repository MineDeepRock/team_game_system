# TeamGameSystem
２チーム以上のPVPゲーム作成を補助するプラグインです。
- 複数ゲームの同時進行
- チーム同士の攻撃キャンセル
- スコアの管理
- 時間制限の管理
- チーム配分
- 人数制限
- Minecraft内でのマップ作成とスポーン地点設定、その他マップ管理
